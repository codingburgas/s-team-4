# Производствена практика 2026 – 10A клас

Това хранилище е стартовият template за курса по Производствена практика.  
Студентите ще работят по двойки върху различни проекти с хардуер, фърмуер, корпус и софтуер.

---

## Структура на проекта


project-template/
├─ firmware/ # Код за микроконтролера и библиотеки (виж firmware/README.md)
├─ hardware/ # Схеми, PCB и Bill of Materials (виж hardware/README.md)
├─ enclosure/ # 3D модели и STL файлове за корпуса (виж enclosure/README.md)
├─ software/ # Код за визуализация и уеб интерфейс (виж software/README.md)
├─ docs/ # План на проекта, изображения и документация (виж docs/README.md)
├─ README.md # Този файл


> Всяка папка съдържа README.md с инструкции какво да се поставя вътре.

---

## Инструкции за работа

1. **Клонирайте репото** чрез GitHub Classroom.  
2. **Не преименувайте репото** — всяка група ще има уникално копие.  
3. Следвайте структурата на папките и поставяйте материалите в съответните директории.
4. Разгледайте поставените на места примерни данни.  
4. **Правете комити и push редовно**, за да се запази прогресът ви.

---

## Насоки за оценяване

Проектите ще се оценяват по:

- Фърмуер  
- Хардуерна част (схеми, PCB, BOM)  
- Проектиране на корпуса (3D модели)  
- Софтуерни компоненти (уеб интерфейс / визуализация)  
- Документация (docs папката и README.md във всяка папка)

---

## Бележки

- Всяка група има **само едно хранилище**.  
- За подробни инструкции вижте README.md във всяка отделна папка.
